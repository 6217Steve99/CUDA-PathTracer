# Caster

CUDA-based physical renderer with editor



## 1. Features

### 1.1 Physical renderer

### Shape

- Line
- Sphere
- Triangle



### Material

- Lambertian
- Mirror
- Dielectric
- Rough Conductor
- substrate



### Tracer

- Path tracing



## 1.2 Editor

- Camera
- Edit by imgui



## 2. Dependencies

- CUDA 11.0
- DX11
